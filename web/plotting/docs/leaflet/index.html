SuperMap iClient Plot for Leaflet API

SuperMap iClient Plot for Leaflet 是一套基于Leaflet的动态标绘，支持访问动态标绘服务，用户可以动态的标绘及编程标绘点标号和线面标号，并能够独立的修改每个标号的风格，也可以编辑标号的大小、位置、形状等。

  • 开发指南：http://iclient.supermap.io/web/introduction/leafletDevelop.html#leafletDynamicPlot
